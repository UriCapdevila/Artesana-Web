# **App Name**: Artesana Online

## Core Features:

- Product Catalog: Display available artisanal jewelry for sale with images and descriptions.
- Shopping Cart: Enable users to add selected jewelry items to a shopping cart.
- Checkout Process: Handle user checkout with Stripe integration.
- AI-Powered Jewelry Recommendations: Suggest other products to a customer using purchase history as a tool to improve sales.
- Order Management: Allow admin to review pending orders, confirm stock availability, mark items as packed and send the item. 
- Inventory Management: A screen allowing to check, edit and adjust stocks for each jewelry product sold on the platform.

## Style Guidelines:

- Primary color: Amber (#A8783C), inspired by the natural and handcrafted quality of the jewelry, evoking warmth and sophistication.
- Background color: Beige (#F5F5DC), a desaturated hue from the primary color, creates a soft and inviting backdrop, enhancing the artisan feel.
- Accent color: Burnt Orange (#CC6633), an analogous hue that complements amber. Its saturation offers visual interest to key UI elements.
- Body text: 'PT Sans', a humanist sans-serif provides readability for product descriptions and general information. Headline font: 'Playfair', a modern serif similar to Didot provides elegance and clear hierarchy
- Lucide-react icons to provide clear and simple visual cues. Used for navigation, social media links, and cart interactions.
- Grid-based layout with Tailwind CSS. Optimizes content spacing. Responsive design for optimal viewing on all devices.
- Subtle animations on product hover and cart interactions to add a layer of refinement and user feedback.
